## Требования

- Java 8 или выше
- JDK для компиляции

## Установка и запуск

### Настройка переменных окружения

Для Windows используйте файл `environment_setup.bat`:
```batch
environment_setup.bat
```

Или настройте вручную:
```batch
set JAVA_HOME=C:\path\to\your\jdk
set PATH=%JAVA_HOME%\bin;%PATH%
```

### Компиляция и запуск

```bash
# Компиляция
javac Passport.java

# Запуск
java Passport
```
